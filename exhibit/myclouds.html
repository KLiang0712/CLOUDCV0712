Self Reflection

Word Clouds are a type of Data Visualization or animation used for analyzing file text.
The objective of Word Cloud building is to read in text and display the most important
Key Words or tags in file text. Tags that appear the most frequently appear in a larger font size
Otherwise, the font size for tags that appear less frequently is reduced if shown at all.
Developing word clouds may also be done with manual programming of various sizes for Key Words of our choice to plan.
Sample 1 is developed using research-based web development and chart CDN scripts.
It is created with basic HTML5, CSS3, and analyzed text saved in the backend (JavaScript).
Embedded in the web development process are JavaScript chart CDN scripts using amCharts and d3.js.
This represents the plug-ins and functionality for displaying the cloud from the backend text.
Based on the backend text analyzed, the tags in the cloud reveal my career interests and what I have learned throughout my studies.
Sample 2 is developed using manual tags in various sizes using basic frontend development with HTML5 and CSS3.
The tags are planned by the developer with larger sizes being the most important and smaller sizes indicating significance as a work-in-progress.
My sample project shows various programming langauges as tags with the larger size being more knowledgeable and
the smaller sized tags as the subjects for self-study or in progress of learning.
