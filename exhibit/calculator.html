    Notes

    This project is subject to occasional updates.


    Percent (%) Button

    Date: 5/20/2024

    New operator added for Remainder division (%) which calculates the remainder of a division in integer form.

    Format for Operators

    Date: 5/20/2024

    Operators: Addition, Subtraction, Multiplication, Division, are in "<"button">" form requiring parseFloat function in JavaScript.

    "Enter =" button

    Date: 5/10/2024

    Evaluates the current expression after pressing it once.
    The expression can be chained with the current output to create a new expression.
    Double-click this button after pushing a button or perform a calculation to clear the textbox.
